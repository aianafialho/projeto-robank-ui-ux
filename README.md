# Robank - UI Design de Aplicativo Bancário

Projeto de interface (UI) de um aplicativo bancário desenvolvido no Figma como estudo de design para aplicações financeiras.

O objetivo foi criar uma experiência moderna, intuitiva e organizada para gestão financeira digital.

Link do Prototipo: https://www.figma.com/proto/NXd52QiAMv5AddntW0Kcs5/projeto_robank?node-id=15-142&starting-point-node-id=1%3A2&t=DjpP33swbruNBTQA-1

## Telas Desenvolvidas

### 1. Tela Inicial (Home)

Tela principal do aplicativo contendo:
- Saudação personalizada
- Saldo disponível
- Acesso rápido a:
  - Pix e Transferência
  - Pagamentos
  - Investimentos
  - Cartões
- Lista de pagamentos recentes

Foco em:
- Hierarquia visual
- Destaque para saldo
- Ações rápidas de uso frequente


### 2. Tela de Extrato

Tela dedicada à visualização detalhada das movimentações financeiras.

Funcionalidades:
- Visualização do saldo disponível
- Saldo separado
- Área de investimentos
- Campo de busca para transações
- Histórico organizado por data contendo:
  - Nome do estabelecimento
  - Horário
  - Tipo de transação
  - Valor

Foco em:
- Clareza das informações
- Organização por blocos
- Legibilidade e contraste

### 3. Tela de Pix e Transferência

Área destinada a pagamentos instantâneos e transferências.

Funcionalidades disponíveis:
- Transferir
- Programar pagamento
- Ler QR Code
- Pix Copia e Cola
- Depositar
- Cobrar

Além de:
- Configurações de Pix
- Cadastro de chaves
- Limites de transferência
- Suporte para contestação

Foco em:
- Acesso rápido
- Ícones intuitivos
- Organização por categorias


## Conceitos de Design Aplicados

- UI moderna para aplicativo financeiro
- Hierarquia visual clara
- Design minimalista
- Uso consistente da cor principal
- Componentização de elementos
- UX focada em simplicidade e eficiência
